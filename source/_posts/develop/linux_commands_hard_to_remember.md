## 一些容易被我忘记的控制台命令
```console
cd -                #返回上一个访问的目录
cat 文件名(|less)    #在终端下查看文件
cp 文件名 目标目录    #将文件拷贝到目标目录下
control+d           #中断a.out运行
```
## 一些浏览器快捷键

``` console
cmd + 鼠标点击       #在新标签页打开点击的链接
```
