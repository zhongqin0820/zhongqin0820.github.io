# 前言
梳理终结英语知识体系中关于语法部分的概念。本文旨在理清基本英语语法脉络，语法即语言的运用法则；英语语法即英语这门语言的运用法则，它研究英语语言的结构规律和使用方法。**英语语法包括：词法和句法两大部分**。

<!-- more -->
# 词法
词法是对词汇的分类，词汇间的搭配关系以及它们在应用中的变化形式进行分析和描述。
## 词性
词性也叫词类，英语单词根据其词义、句法作用和形式特征进行分类，可以分成十个大类。再根据是否可以构成句子的一个成分将十大类分为实词与虚词。

### 实词
- [名词](https://www.hjenglish.com/cixing/mingciyingyu/)；noun；n.
    - 表示人或事物的名称
    - 可数，不可数
- [代词](https://www.hjenglish.com/cixing/yingyudaici/)；pronoun；pron.
    - 代替名词的词
    - 人称代词，物主代词，反身代词，相互代词，指示代词，疑问代词，关系代词，连接代词，**不定代词**
- [形容词](https://www.hjenglish.com/cixing/yyxrc/)；adjective；adj.
    - 修饰名词等，说明事物或人的性质或特征的词位置不一定都放在名词前面
    - 性质形容词，叙述形容词
- [副词](https://www.hjenglish.com/cixing/yingyufuci/)；adverb；adv.
    - 主要用来修饰动词，形容词，副词或整个句子
    - 时间副词，地点副词，程度副词方式副词，频度副词
- [动词](https://www.hjenglish.com/cixing/dongciyingyu/)；verb；v.
    - 表示动作或状态
    - 行为动词（及物，不及物），系动词（be），助动词和情态动词
- [数词](https://www.hjenglish.com/cixing/yysc/)；numeral；num.
    - 表示数目，表示顺序
    - 基数词，序数词

### 虚词
|中文|英文|缩写|作用|分类|举例|
|:---:|:---:|:---:|:---|:---|:---|
|[冠词](https://www.hjenglish.com/cixing/yingyuguanci/)|article|art.|用在名词前，帮助说明名词所指的范围|定冠词，不定冠词|a 一个|
|[介词](https://www.hjenglish.com/cixing/yyjc/)|preposition|prep.|用在名词或代词前，说明它与别的词的关系|简单介词，合成介词<br>成语介词，分词介词|at 在...|
|[连词](https://www.hjenglish.com/cixing/yylc/)|conjunction|conj.|用于连接单词，短语，从句或句子|并列连词（转折，因果，并列）<br>从属连词|and 和|
|[感叹词](https://www.hjenglish.com/new/p144076/)|interjection|interj.|用来表示说话时表达的喜怒哀乐等情感的词|基础感叹词，其它此类作感叹词|oh 哦|

## 动词
在后文句法部分，我们知道一个最基本的简单句可以由`主语+谓语`构成。谓语作为一个句子的核心，其对句子构成的影响意义可见一斑。而动词是构成谓语成分的重要组成，因此，这里单独对动词展开介绍。
### 谓语动词的时态/语态
一个句子的时态/语法常受谓语动词的时态/语法影响。
#### 时态
- 一般（习惯性）
    - 过去：did（做了）
    - 现在：do（通常做）
    - 将来：will do/ be going to do（将要做）
- 进行（正在发生）
    - 过去：was/were doing（当时正在做）
    - 现在：am/is/are doing（现在正在做）
    - 将来：will be doing/be going to be doing（将来正在做）
- 完成（有结果）
    - 过去：I had finished it by yesterday
    - 现在：I have finished it since yesterday
    - 将来：I will have finished it by tomorrow
- 完成进行（持续性：一直在做）
    - 过去：I had been working on it before yesterday
    - 现在：I have been working ono it since yesterday
    - 将来：I will have been working on it before tomorrow

#### 语态
- 主动
- 被动：be动词 + 动词过去分词（注意：过去分词≠过去式）

### 非谓语动词
非谓语动词除了不能独立作谓语外，可以承担句子的其他成分。主要包括：
- 不定式
- 动名词
- 分词
    - 现在分词：和主句主语为主动关系；注意与动名词做区分。
    - 过去分词：和主句主语为被动关系

# 句法
句法是对句子的各种类型，各种句子成分以及遣词造句的规律等进行分析和探讨。

## 句子成分
一个完整的英文句子由不同成分构成；英语句子成分分为七种：主语，谓语，宾语，定语，状语，表语，宾语补足语。

|中文|英文|缩写|作用|涉及词性|
|:---:|:---:|:---:|:---|:---|
|主语|subject|S|全局述说的对象|名词，代词或相当于名词的词组或句子充当|
|谓语|predicate|V|主语做的动作，状态或特征|动词或动词短语|
|宾语|object|O|动作作用的对象|名词|
|宾语补足语|object complement|OC|对宾语进行补充和说明，使其意义完整|形容词，不定式（to do）|
|表语|predicative|P|也称为主语的补语，表示主语的身份，性质，状态和特征|形容词，不定式（to do）|
|定语|attribute|attrib.|修饰限定名词或代词|形容词，短语或从句|
|状语|adverbial|adv.|修饰作用，表状态|副词|

## 句型
这些成分构成的句子从结构上可以大致分为以下三大类。下文重点展开介绍简单句以及从句的基本构成。
- 简单句
- 并列句：简单句+（并列词）+简单句
- 复合句：简单句+（从句引导词）+简单句

### 简单句
简单句主要有6个基本句型。
- S + V + O
- S + V
- S + V + P
- S + V + o + O
- S + V + O + C
- there be
    - there be + 名词
    - there be + 名词 + 介宾

### 从句
从句的一般构成形式为：`从句 = 引导词 + S + V + O`；根据从句的类型可以大致分为三类：
- 名词性从句
- 定语从句（形容词性）从句
- 状语（副词性）从句

#### 名词性从句
又根据在句子中作不同的成分，分为：
- 主语从句
- 宾语从句
- 表语从句
- 同位语从句

#### 定语从句（形容词性）从句
- 作用：修饰限定名词

#### 状语（副词性）从句
- 状语从句：起副词作用（修饰动词，形容词）的句子
- 按作用分类：
    - 时间，地点，原因，条件
    - 目的，结果，让步，方式

## 特殊句式
- 强调句
- 倒装句：强调，加强语气的作用
    - 正常句：S+V+(O)
    - 倒装句：V+S+(O)
- 虚拟语气：假设一件有可能发生的事
