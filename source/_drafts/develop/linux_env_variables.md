# 环境变量
环境变量是指系统环境变量，对所有用户起作用，而用户环境变量只对当前用户起作用

由于一般情况下，我们的电脑只有一个主用户，所以在其中任何一个配置即可，当然以防万一你也可以只在系统环境变量下配置。 

用户变量和系统变量的区别在于权限和实例。

系统变量则不同，任何时候都只有一套，不会随用户登录状况发生变化。
